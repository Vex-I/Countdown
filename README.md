# Countdown
A dead simple countdown app made with ``React``. Made to measure the hours between now and the target date. Features an activity trackers 
to track time spent on a given activity, and productivity bar that goes up the more time you spend working in a day, for up to 12 hours.
Data is stored in the ``localStorage`` of the browser.

Quick Start
---
To run locally, simply go to ``chrome://extensions/`` load the ``build`` directory of this repository, and run the extension. You can update the 
build by running ``npm run build`` inside the main directory.
